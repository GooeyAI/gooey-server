{# # Template guide ## Context required: - settings.PAYPAL_CLIENT_ID: str - settings.APP_BASE_URL: str - payment_processing_url: str ## Usage: PayPal One-Time 1. Create a div with `id="paypal-addon-buttons"` where you want to render the PayPal One-Time buttons. 2. Use the JavaScript function `setPaypalAddonQuantity(qty)` to set the number of credits to purchase. Whenever the user clicks on the one-time button rendered by PayPal, they will be redirected to PayPal to buy `qty` credits. ## Usage: PayPal subscriptions 1. Create a div with `class="paypal-subscription-buttons"` and `data-plan-id="the plan id"` where you want to render the subscribe button for that plan. Whenever user clicks on the subscribe button rendered by PayPal, they will be redirected to PayPal to subscribe to that plan. #}
